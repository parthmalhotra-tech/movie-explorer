# 🎬 Movie Explorer

A full-stack Movie Explorer web application built using FastAPI, HTML, CSS, Jinja2, SQLite, and SQLAlchemy.

## Features

- 🔍 Search movies using an external Movie API
- 🎬 View movie details
- 👤 User Authentication (Login & Signup)
- 👤 User Profile
- 📱 Responsive UI
- 💾 SQLite Database
- ⚡ FastAPI Backend

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- Jinja2

### Frontend
- HTML5
- CSS3

## Installation

```bash
git clone https://github.com/parthmalhotra-tech/movie-explorer.git

cd movie-explorer

pip install -r requirements.txt

uvicorn app:app --reload
```

Open your browser:

```
http://127.0.0.1:8000
```
